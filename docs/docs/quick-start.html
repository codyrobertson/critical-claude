üöÄ Quick Start Guide

Get up and running with Critical Claude in under 5 minutes

Prerequisites

Before installing Critical Claude, ensure you have the following:

  • Node.js 18+ - Latest LTS version recommended
  • npm 9+ - For workspace and MCP support
  • Git - For cloning the repository
# Check your versions
node --version  # Should be 18.0.0 or higher
npm --version   # Should be 9.0.0 or higher
git --version   # Any recent version

Installation Methods

Choose the installation method that best fits your workflow:

Local Development Setup

Best for development and customization:

# 1. Clone the repository
git clone https://github.com/critical-claude/critical-claude.git
cd critical-claude

# 2. Install dependencies and build
npm run setup

# 3. Start all services
npm run start:all

NPM Package Installation

Quick global installation:

# Install globally
npm install -g critical-claude

# Verify installation
critical-claude --version

# Setup for your editor
critical-claude setup claude-code

Docker Installation

Isolated container environment:

# Pull the image
docker pull critical-claude/critical-claude:latest

# Run with MCP server
docker run -d -p 3000:3000 \
  -v $(pwd):/workspace \
  critical-claude/critical-claude:latest

      "command": "node",
      "args": ["./packages/code-critique/dist/server.js"],
      "cwd": "/path/to/critical-claude"

}

Troubleshooting

Common issues and solutions:

‚ùå MCP Server Not Loading

Symptoms: Tools not available in Claude Code

Solution:

# Check server is running
node ./packages/code-critique/dist/server.js

# Verify path in settings.json
# Restart Claude Code

‚ùå Build Errors

Symptoms: npm run setup fails

Solution:

# Clean and rebuild
npm run clean
npm run setup

# Check Node.js version
node --version  # Must be 18+

‚ùå Permission Errors

Symptoms: EACCES errors during installation

Solution:

# Fix npm permissions
npm config set prefix ~/.npm-global
export PATH=~/.npm-global/bin:$PATH

# Or use npx instead of global install
npx critical-claude setup claude-code
