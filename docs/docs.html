Installation

Global Installation (Recommended)

npm install -g critical-claude

Verify Installation

cc --version

Local Development

git clone https://github.com/codyrobertson/critical-claude.git cd critical-claude npm install npm run build
Note: Node.js 18+ is required for optimal performance.

Configuration

Environment Setup

Create a cc.env file in your project root:

# Option 1: OpenAI (Recommended for most users) OPENAI_API_KEY=sk-your-openai-api-key-here # Option 2: Anthropic Claude API ANTHROPIC_API_KEY=sk-ant-your-anthropic-api-key-here # Project Settings (Optional) CC_PROJECT_NAME=my-project CC_TEAM_SIZE=5 CC_EXPERIENCE_LEVEL=senior

Claude Desktop Integration

Add to ~/.claude/settings.json:

{ "mcpServers": { "critical-claude": { "command": "cc", "args": ["mcp-server"], "env": { "NODE_ENV": "production" } } } }
Security Note: Never commit API keys to version control. Add cc.env to your .gitignore.

Basic Usage

Initialize Project

cc init

Create Tasks

cc task create "Implement user authentication"

List Tasks

# List all tasks cc task list # Filter by status cc task list --status in_progress # Filter by priority cc task list --priority high

Update Tasks

# Change status cc task edit task-123 --status done # Assign to team member cc task edit task-123 --assignee alice # Add labels cc task edit task-123 --labels "backend,api"

View Task Details

cc task view task-123

AI Features

AI Task Generation

cc task ai "Build e-commerce checkout flow"

Generates 8-12 related tasks with dependencies automatically.

Task Expansion

# Expand a complex task into subtasks cc task expand task-123 # Control expansion parameters cc task expand task-123 --maxTasks 8 --teamSize 3

Multi-Agent Research

cc task research "Implement real-time notifications"

Four specialized agents analyze and create actionable implementation tasks:

  • Planner: Breaks down requirements
  • Researcher: Gathers technical information
  • Analyst: Evaluates complexity and risks
  • Writer: Creates actionable tasks

Effort Estimation

cc task estimate task-123 --apply

Provides story points, hours estimate, and confidence level.

Claude Desktop Integration

MCP Server

Critical Claude provides native Model Context Protocol (MCP) integration:

# Start MCP server cc mcp-server

Available MCP Tools

  • task_create: Create new tasks
  • task_list: List and filter tasks
  • task_update: Update task properties
  • task_ai_generate: Generate tasks with AI
  • template_load: Load project templates
Tip: Use Claude Desktop to naturally interact with your tasks using conversational language.

Advanced Usage

Bulk Operations

# Archive completed tasks cc task list --status done | xargs -I {} cc task archive {} # Update multiple tasks cc task edit --status in_progress --assignee alice task-1 task-2 task-3

Export/Import

# Backup tasks cc task backup # Export to JSON cc task export --format json > tasks.json # Import from file cc task import tasks.json

Task Viewer

cc task view-ui

Interactive terminal UI with:

  • Real-time task updates
  • Vim-style keybindings
  • Fuzzy search
  • Task detail views

Troubleshooting

Common Issues

AI Service Not Working

# Check configuration cc config check # Verify API key echo $OPENAI_API_KEY # Test AI connection cc task ai "test task" --dry-run

Template Loading Errors

# Validate template syntax cc task template validate webapp # Reset templates cc task template reset

Dependency Conflicts

# Force status change (use carefully) cc task edit task-123 --status done --force # Clear all dependencies cc task edit task-123 --clear-dependencies

Debug Mode

# Enable verbose logging CC_DEBUG=true cc task ai "debug test" # Check system info cc --info
