cc init
Initialize a new Critical Claude project in the current directory.
cc init [options]
Option Description Default
--name Project name Current directory name
--template Initial template to load None
--team-size Team size (affects AI task generation) 3
Basic Initialization
Initialize project with default settings
cc init
With Template
Initialize and load a project template
cc init --template webapp --name my-project
cc task
Core task management commands for creating, listing, editing, and organizing tasks.

Subcommands

cc task create

cc task create <title> [options]
Option Description Example
--priority Task priority (critical, high, medium, low) --priority high
--assignee Assign to team member --assignee alice
--labels Comma-separated labels --labels "backend,api"
--story-points Effort estimation --story-points 5
--dependencies Task dependencies --dependencies task-123

cc task list

cc task list [options]
Option Description Example
--status Filter by status --status in_progress
--priority Filter by priority --priority critical
--assignee Filter by assignee --assignee alice
--labels Filter by labels --labels backend
--format Output format (table, json, csv) --format json

cc task edit

cc task edit <task-id> [options]
Option Description Example
--status Change status --status done
--title Update title --title "New title"
--force Force change (bypass dependency checks) --force
Natural Language Parsing
Create tasks using natural language
cc task create "Fix login bug @critical #security 5pts for:alice"
Bulk Operations
Update multiple tasks at once
cc task edit --status in_progress task-1 task-2 task-3
cc task ai
Generate comprehensive task breakdowns using AI from natural language descriptions.
cc task ai <description> [options]
Option Description Default
--maxTasks Maximum number of tasks to generate 8
--teamSize Target team size 3
--experience Team experience level (junior, intermediate, senior) intermediate
--dry-run Preview without creating tasks false
Basic AI Generation
Generate tasks for a web application
cc task ai "Build user authentication with OAuth2 and JWT tokens"
Advanced Generation
Fine-tune generation parameters
cc task ai "E-commerce checkout flow" --maxTasks 12 --teamSize 5 --experience senior

cc task expand
Expand a single complex task into multiple smaller, manageable subtasks.
cc task expand <task-id> [options]
Option Description Default
--maxTasks Maximum number of subtasks 8
--granularity Task granularity (coarse, medium, fine) medium
--preserve-original Keep original task as parent true
Task Expansion
Break complex task into subtasks
cc task expand task-123 --maxTasks 6 --granularity fine
cc viewer
Launch interactive terminal UI for task management with real-time updates and vim-style navigation.
cc viewer [options]
Option Description Default
--theme UI theme (dark, light, auto) auto
--layout Layout mode (split, full) split
--filter Initial filter none

Vim-style Keybindings:

  • j/k: Navigate up/down
  • h/l: Navigate left/right
  • /: Search tasks
  • n/N: Next/previous search result
  • Enter: View task details
  • e: Edit task
  • d: Mark task as done
  • q: Quit viewer
