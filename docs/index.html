Get Started in Minutes

Choose your development environment and start analyzing code immediately

1

Install Critical Claude

Clone the repository and set up the development environment

git clone https://github.com/codyrobertson/critical-claude.git cd critical-claude npm run setup
2

Configure MCP Server

Add Critical Claude to your Claude Code MCP settings

{ "mcpServers": { "critical-claude": { "command": "node", "args": ["./packages/code-critique/dist/server.js"], "cwd": "/path/to/critical-claude" } } }
3

Start analyzing code

Use Critical Claude tools directly in your Claude Code conversations

Use cc_crit_code to analyze this function for security vulnerabilities: function authenticateUser(username, password) { return users.find(u => u.username === username && u.password === password); }
