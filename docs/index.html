Quick Installation

Get up and running in under 5 minutes

1

Clone Repository

git clone https://github.com/critical-claude/critical-claude.git
cd critical-claude
2

Build & Setup

npm run setup
3

Configure Claude Code

Add to your Claude Code settings:

{
  "mcpServers": {
    "critical-claude": {
      "command": "node",
      "args": ["./packages/code-critique/dist/server.js"],
      "cwd": "/path/to/critical-claude"
    }
  }
}
1

Add .cursorrules

Copy the Critical Claude persona to your project:

curl -o .cursorrules https://raw.githubusercontent.com/critical-claude/critical-claude/main/.cursorrules
2

Enable Battle-Tested Mode

Cursor will automatically apply the Critical Claude persona to all code analysis

1

Configure Assistant

Add to your Zed settings:

{
  "assistant": {
    "version": "2",
    "default_model": {
      "provider": "anthropic",
      "model": "claude-3-5-sonnet-20241022"
    }
  }
}
2

Import Critical Claude Context

Use the assistant with Critical Claude instructions for code analysis

Install Extension

2

Configure Settings

Enable battle-tested analysis in extension settings
