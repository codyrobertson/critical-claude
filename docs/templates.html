Using Templates

üìã List All Templates

cc task template list

View all available templates with descriptions and task counts.

üëÄ Preview Template

cc task template show webapp

Examine template structure and variables before loading.

üöÄ Load Template

cc task template webapp --framework react

Load template with custom variables and create tasks.

üîß Customize Variables

cc task template api --framework express --database mongodb --auth jwt

Override default variables to match your project needs.

Template Customization

Common Variables

Variable Description Examples Templates
project_name Name of your project my-app, api-service All templates
framework Primary framework to use react, vue, express webapp, api, mobile-app
database Database system postgres, mongodb, mysql webapp, api, microservice
auth_type Authentication method jwt, oauth2, session webapp, api
ml_framework ML framework pytorch, tensorflow machine-learning
deployment_target Deployment platform docker, kubernetes, aws webapp, api, microservice

Creating Custom Templates

# Create new template cc task template create my-template # Edit template file # Templates are stored in TOML format with metadata and tasks

Template Inheritance

Templates can inherit from other templates, allowing you to build on existing structures:

# In your custom template TOML file [metadata] inherits = "api" name = "my-custom-api" # Override or add tasks as needed
